animate

  • animate(selection: any, attrToProjector: IAttributeToProjector, plot: Plot): any
    • Defined in plottable.d.ts:1136

    Applies the supplied attributes to a D3.Selection with some animation.

    Parameters

    • selection: any

      The update selection or transition selection that we wish to animate.

    • attrToProjector: IAttributeToProjector

      The set of IAccessors that we will use to set attributes on the selection.

    • plot: Plot

      The plot being animated.

    Returns any

    Animators should return the selection or transition object so that plots may chain the transitions between animators.
