    Converts an HSL color value to RGB. Conversion formula adapted from http://en.wikipedia.org/wiki/HSL_color_space. Assumes h, s, and l are contained in the set [0, 1] and returns r, g, and b in the set [0, 255]. Source: https://stackoverflow.com/questions/2353211/hsl-to-rgb-color-conversion
