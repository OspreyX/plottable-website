attr

  • attr(attrToSet: string, accessor: any, scale?: Scale): Plot

  • Inherited from Plot.attr

    • Defined in plottable.d.ts:2234

    Sets an attribute of every data point.

    Here's a common use case:

    plot.attr("r", function(d) { return d.foo; });

    This will set the radius of each datum d to be d.foo.

    Parameters

    • attrToSet: string

      The attribute to set across each data point. Popular examples include "x", "y", "r". Scales that inherit from Plot define their meaning.

    • accessor: any

      Function to apply to each element of the dataSource. If a Function, use accessor(d, i). If a string, d[accessor] is used. If anything else, use accessor as a constant across all data points.

    • scale?: Scale optional

      If provided, the result of the accessor is passed through the scale, such as scale.scale(accessor(d, i)).

    Returns Plot

    The calling Plot.

merge

  • merge(c: Component): Group

  • Inherited from Component.merge

    • Defined in plottable.d.ts:1553

    Merges this Component with another Component, returning a ComponentGroup. This is used to layer Components on top of each other.

    There are four cases: Component + Component: Returns a ComponentGroup with both components inside it. ComponentGroup + Component: Returns the ComponentGroup with the Component appended. Component + ComponentGroup: Returns the ComponentGroup with the Component prepended. ComponentGroup + ComponentGroup: Returns a new ComponentGroup with two ComponentGroups inside it.

    Parameters

    • c: Component

      The component to merge in.

    Returns Group

    The relevant ComponentGroup out of the above four cases.
