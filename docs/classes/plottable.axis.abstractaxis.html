    Merges this Component above another Component, returning a ComponentGroup. This is used to layer Components on top of each other.

    There are four cases: Component + Component: Returns a ComponentGroup with the first component after the second component. ComponentGroup + ComponentGroup: Returns a new ComponentGroup with the first group after the second group.

    Parameters
