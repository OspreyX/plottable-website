addComponent

  • addComponent(row: number, col: number, component: AbstractComponent): Table
    • Defined in plottable.d.ts:2506

    Adds a Component in the specified cell. The cell must be unoccupied.

    For example, instead of calling new Table([[a, b], [null, c]]), you could call

    var table = new Table();
table.addComponent(0, 0, a);
table.addComponent(0, 1, b);
table.addComponent(1, 1, c);

    Parameters

    • row: number

      The row in which to add the Component.

    • col: number

      The column in which to add the Component.

    • component: AbstractComponent

      The Component to be added.

    Returns Table

    The calling Table.

merge

  • merge(c: AbstractComponent): Group

  • Inherited from AbstractComponent.merge

    • Defined in plottable.d.ts:1755

    Merges this Component with another Component, returning a ComponentGroup. This is used to layer Components on top of each other.

    There are four cases: Component + Component: Returns a ComponentGroup with both components inside it. ComponentGroup + Component: Returns the ComponentGroup with the Component appended. Component + ComponentGroup: Returns the ComponentGroup with the Component prepended. ComponentGroup + ComponentGroup: Returns a new ComponentGroup with two ComponentGroups inside it.

    Parameters

      The component to merge in.

    Returns Group

    The relevant ComponentGroup out of the above four cases.

rowWeight

  • rowWeight(index: number, weight: number): Table
    • Defined in plottable.d.ts:2543

    Sets the layout weight of a particular row. Space is allocated to rows based on their weight. Rows with higher weights receive proportionally more space.

    A common case would be to have one row take up 2/3rds of the space, and the other row take up 1/3rd.

    Example:

    plot = new Plottable.Component.Table([
 [row1],
 [row2]
]);

// assign twice as much space to the first row
plot
 .rowWeight(0, 2)
 .rowWeight(1, 1)

    Parameters

    • index: number

      The index of the row.

    • weight: number

      The weight to be set on the row.

    Returns Table

    The calling Table.
