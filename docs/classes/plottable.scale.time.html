autoDomain

  • autoDomain(): AbstractScale

  • Inherited from AbstractScale.autoDomain

    • Defined in plottable.d.ts:905

    Modifies the domain on the scale so that it includes the extent of all perspectives it depends on. This will normally happen automatically, but if you set domain explicitly with plot.domain(x), you will need to call this function if you want the domain to neccessarily include all the data.

    Extent: The [min, max] pair for a Scale.Quantitative, all covered strings for a Scale.Category.

    Perspective: A combination of a Dataset and an Accessor that represents a view in to the data.
