Class Broadcaster

The Broadcaster class is owned by an Listenable. Third parties can register and deregister listeners from the broadcaster. When the broadcaster.broadcast method is activated, all registered callbacks are called. The registered callbacks are called with the registered Listenable that the broadcaster is attached to, along with optional arguments passed to the broadcast method.

The listeners are called synchronously.
